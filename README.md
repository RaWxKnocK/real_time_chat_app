# 💬 Real-Time Chat App

A simple real-time chat application built using modern technologies that allows users to send and receive messages instantly.

---

## 🚀 Features

* 🔥 Real-time messaging
* 👤 Multiple users support
* ⚡ Instant message delivery
* 🌐 WebSocket-based communication
* 📱 Simple and responsive UI

---

## 🛠️ Tech Stack

* **Backend:** Spring Boot
* **Frontend:** HTML, CSS, JavaScript
* **Real-time Communication:** WebSockets / STOMP

---

## ⚙️ Setup & Run

### 1. Clone the repository

```bash
git clone https://github.com/RaWxKnocK/real_time_chat_app.git
cd real_time_chat_app
```

### 2. Open in IntelliJ

* Open project
* Wait for Maven dependencies to load

### 3. Run the application

* Run main Spring Boot class

### 4. Open in browser

```
http://localhost:8080/chat
```
